## 📖 Overview

**Indian Temple Heritage and Pilgrimage** is a comprehensive web platform that showcases India's ancient temples, sacred pilgrimage circuits, spiritual traditions, and architectural heritage.

The platform serves as a one-stop destination for devotees, tourists, researchers, and heritage enthusiasts to explore temples across India, learn their history, discover pilgrimage routes, and understand their cultural significance.

---

## ✨ Features

- 🛕 Explore famous temples across India
- 📍 Temple information with location and history
- 🏛️ Detailed architectural and cultural insights
- 🧭 Pilgrimage routes and sacred circuits
- 🗺️ Browse temples state-wise and region-wise
- 📷 High-quality temple galleries
- 🔍 Powerful search functionality
- ❤️ Save favorite temples
- 📱 Fully responsive interface
- ⚡ Fast loading and optimized performance

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/indian-temple-heritage-and-pilgrimage.git
```

### Navigate to the Project

```bash
cd indian-temple-heritage-and-pilgrimage
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
